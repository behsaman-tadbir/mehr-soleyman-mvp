# Playful EdTech Color Palette — Mehr Soleyman MVP

این پالت برای نسخه MVP دمو پروژه مهر سلیمان است و با تمرکز روی حس آموزشی، شاد و خوانا تدوین شده است.

## رنگ‌ها و کدهای HEX
- **Primary**: `#38BDF8`
- **Secondary**: `#4ADE80`
- **CTA** (Call To Action): `#FB923C`
- **Highlight**: `#A78BFA`
- **Background**: `#F8FAFC`
- **Text**: `#111827`
- **Muted Text**: `#6B7280`
- **Border**: `#E2E8F0`

## نقش و کاربرد هر رنگ
- **Primary (#38BDF8)**: المان‌های کلیدی برند، لینک‌ها و المان‌های تعاملی ثانویه.
- **Secondary (#4ADE80)**: وضعیت‌های موفق یا مثبت، برچسب‌های حمایتی و اکنت‌های ملایم.
- **CTA (#FB923C)**: فقط برای دکمه‌ها و اقدام‌های اصلی. جایگزین ندارد.
- **Highlight (#A78BFA)**: تاکید محدود روی بخش‌های کوچک (بدون استفاده برای متن‌های طولانی).
- **Background (#F8FAFC)**: رنگ پس‌زمینه عمومی صفحات.
- **Text (#111827)**: متن‌های اصلی و تیترها.
- **Muted Text (#6B7280)**: توضیحات و متن‌های ثانویه/کم‌رنگ.
- **Border (#E2E8F0)**: خطوط جداکننده و مرزبندی ظریف.

## قوانین استفاده
- **CTA فقط برای اقدام‌ها**: هیچ متن عادی یا لینک ساده نباید با CTA استایل شود.
- **Highlight محدود**: فقط برای تگ‌های کوچک، آیکون‌های تأکیدی یا بخش‌های کوتاه.
- **متن هرگز با Primary/Secondary**: رنگ متن اصلی همیشه Text یا Muted Text باشد.
- **پس‌زمینه و باکس‌ها**: از Background برای سطح پایه و از Border برای خطوط ظریف استفاده شود.

## نکات دسترس‌پذیری
- نسبت کنتراست متن با پس‌زمینه باید حداقل 4.5:1 برای متن معمولی و 3:1 برای تیترهای بزرگ باشد.
- دکمه‌های CTA باید در حالت Hover/Active نیز کنتراست کافی با پس‌زمینه حفظ کنند.
- برای متن روی CTA یا Primary از رنگ Text یا سفید با نسبت کنتراست مناسب استفاده شود.
- هایلایت‌ها نباید خوانایی متن را کاهش دهند؛ در صورت استفاده، کنتراست متن سنجیده شود.
